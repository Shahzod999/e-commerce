nodemon
Используется для автоматической перезагрузки сервера при внесении изменений в код. Это облегчает разработку, так как вам не нужно вручную перезапускать сервер каждый раз после изменения кода.

multer
Middleware для обработки файлов, отправляемых через форму. Позволяет загружать файлы на сервер и управлять ими, например, сохранять на диск или в базу данных.

mongoose
Библиотека для работы с MongoDB в Node.js. Предоставляет удобный интерфейс для создания моделей данных, выполнения CRUD-операций, и валидации данных.

jsonwebtoken
Библиотека для работы с JSON Web Token (JWT). Используется для создания и проверки токенов, которые обычно используются для аутентификации и авторизации пользователей.

express-formidable
Middleware для обработки форм данных (включая файлы) в Express-приложениях. Поддерживает форматы multipart/form-data и urlencoded.

express-async-handler
Middleware для упрощения обработки асинхронных операций в Express. Позволяет писать асинхронные функции в маршрутах, обрабатывая ошибки автоматически.

express
Фреймворк для создания веб-приложений и API на Node.js. Обеспечивает базовые функции для маршрутизации, обработки запросов и ответов, использования middleware, и многое другое.

dotenv
Библиотека для работы с переменными окружения. Позволяет хранить конфиденциальные данные (например, ключи API, пароли) в файле .env и загружать их в приложение.

cors
Middleware для управления политикой Cross-Origin Resource Sharing (CORS). Позволяет вашему серверу обрабатывать запросы из других доменов, что необходимо для создания API, доступных из различных источников.

cookie-parser
Middleware для работы с cookies в Express-приложениях. Позволяет парсить и устанавливать cookies в HTTP-запросах.

concurrently
Утилита для запуска нескольких команд одновременно. Обычно используется для запуска нескольких процессов, например, сервера и фронтенд-приложения, в одном терминале.

bcryptjs
Библиотека для хеширования паролей. Используется для обеспечения безопасности паролей, хранящихся в базе данных, путем их хеширования и проверки на соответствие при входе пользователя.
